# Data Directory

This directory contains input data for inference and testing.

## Structure

- `images/input/` - Input images for inference
- `images/test/` - Test images for validation
- `videos/input/` - Input videos for inference (optional)

## Supported Formats

- **Images**: JPG, PNG, BMP, TIFF
- **Videos**: MP4, AVI, MOV, MKV

## Usage

Place your input images or videos in the respective directories before running inference scripts.
